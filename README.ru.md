# Анализ продаж Superstore

[English](README.md) | Русский

Проект по исследовательскому анализу данных розничных продаж с использованием Python, pandas, seaborn и Jupyter. Цель проекта - показать, как из табличных данных можно получить понятные бизнес-выводы о выручке, прибыли, регионах и убыточных заказах.

## О проекте

В проекте анализируется датасет `Sample - Superstore.csv`, чтобы понять:

- какие категории приносят наибольшую прибыль
- какие регионы лидируют по продажам
- где компания теряет деньги
- какие товары дают наибольшую выручку

Выполнены следующие шаги:

- загрузка и проверка данных
- проверка пропусков и дубликатов
- расчёт продаж и прибыли
- анализ категорий и регионов
- поиск убыточных строк заказов
- визуализация результатов

## Бизнес-вопросы

- Какие категории товаров самые прибыльные?
- Какие регионы лидируют по продажам?
- Сколько в датасете убыточных строк заказов?
- Какие товары приносят наибольшую выручку?
- Какие закономерности могут объяснить слабую прибыльность?

## Стек

- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Датасет

- Исходный файл: `Sample - Superstore.csv`
- Предметная область: розничные продажи
- Основные поля: `Sales`, `Profit`, `Category`, `Region`, `Product Name`

## Основные выводы

- Общая выручка: `2,297,200.86`
- Общая прибыль: `286,397.02`
- Самая прибыльная категория: `Technology`
- Регион с наибольшими продажами: `West`
- Количество убыточных строк заказов: `1,871`

Эти результаты показывают, что высокая выручка не всегда означает высокую прибыль, поэтому в дальнейшем стоит глубже изучить скидки, состав продуктового портфеля и причины убыточных продаж.

## Как запустить

1. Склонировать репозиторий.
2. Установить зависимости:

```bash
pip install pandas matplotlib seaborn jupyter
```

3. Запустить Python-скрипт:

```bash
python analysis.py
```

4. Или открыть ноутбук:

```bash
jupyter notebook notebook.ipynb
```
